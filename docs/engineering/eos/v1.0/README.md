# 03_Release_Package

## Purpose

This folder contains the official release package for each published version of the **Team LIPS Engineering Operating System (EOS)**.

The Release Package provides the supporting documents that accompany the official Engineering Operating System specification and communicates the release to leadership, engineering teams, AI workspaces, and future Team LIPS products.

---

## Contents

Typical contents include:

- Executive Summary
- Quick Start Guide
- Release Notes
- Changelog
- Document Manifest
- Approval Record
- Release Certificate
- Future release documentation

---

## Release Workflow

Engineering releases follow the approved lifecycle.

```text
Draft
  │
  ▼
Engineering Review
  │
  ▼
Mission Control Review
  │
  ▼
Founder Approval
  │
  ▼
Published
  │
  ▼
Release Package
  │
  ▼
GitHub Publication
  │
  ▼
Engineering Workspace Synchronization
```

Only **published** releases shall be stored in this folder.

## What Belongs Here

- Official Executive Summaries
- Official Quick Start Guides
- Official Release Notes
- Official Changelogs
- Official Approval Records
- Official Document Manifests
- Official Release Certificates
- Published release documentation

---

## What Does NOT Belong Here

- Working drafts
- Review copies
- Engineering discussions
- Mission packages
- Verification reports
- Build prompts
- Temporary files

These belong in their respective workflow folders.
